DEXNAV

¡Busca el nombre o identificador de cualquier Pokémon!


  • Los Pokémon son una clase de criaturas inspiradas en animales reales, insectos, objetos, plantas o criaturas mitológicas.



  • Es una enciclopedia virtual portátil de alta tecnología que los entrenadores Pokémon llevan consigo para registrar las fichas de todas las diversas especies Pokémon que existen.



  • Existen un total de 898 Pokémon. ¡Comprueba cuál es el último!



  • Esta aplicación web se desarrolló utilizando la PokeApi.
